# Hotels app

## Installation
```
$ git clone git@github.com:trepafi/testing-gulp-angular.git test
$ cd test
$ npm install
$ bower install
$ ./node_modules/gulp/bin/gulp.js serve
```

## Test it
Open your browser and access [localhost:3000](http://localhost:3000)
